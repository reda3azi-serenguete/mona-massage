# 5-ELEMNTS

A modern React + TypeScript web application built with Vite, Tailwind CSS, and ESLint.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
src/
  ├── components/        # React components
  ├── pages/            # Page components
  ├── lib/              # Utility functions
  ├── styles/           # Global styles
  ├── App.tsx           # Root component
  └── main.tsx          # Entry point
```

## 🛠️ Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **ESLint** - Code quality

## 📝 Development

This project uses the **Frontend Code Reviewer** agent for production-level code analysis. Use it to:
- Review code quality and performance
- Check TypeScript type safety
- Audit CSS and Tailwind patterns
- Identify UX/UI inconsistencies

## 📚 Scripts

- `npm run dev` - Start development server (http://localhost:5173)
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
